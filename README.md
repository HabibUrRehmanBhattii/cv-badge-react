# Intro React App with Vite

This is a simple React app created with Vite.

# Live Demo

[Netlify](https://lucent-gnome-a23bfd.netlify.app/)


## Getting Started

Install dependencies:

```bash
npm install
```

Create Vite App:

```bash
npm init @vitejs/app
```

Run the dev server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```
